星灯りの巡礼路と灰眠りの竜

年に一度、山上の空鐘修道院で「星護の鐘灯」を灯すことで、谷に眠る灰竜は静かに夜を越える。 だが今年、その灯は壊れ、継ぎ手である老灯守は息絶えた。残された見習いの少女リネットと、 彼女を助ける三人の冒険者は、夜明けまでに修道院へ辿り着き、鐘灯を継がねばならない。 旅の魅力、共同作業、そして少しだけ変で美しい逸脱を前に出した、王道JRPG寄りの単発シナリオである。

このシナリオで前に出すもの

  • 安全ルート / 近道 / 地元ルートの選択
  • 村と山道と修道院それぞれの文化差・通行差
  • 同行NPCを守るだけでなく、彼女と一緒にやるから通る工程
  • 「変で美しい」現象を、危険無効化ではなく情景と報酬へ変換する余地
  • 極の死亡許容度に見合う、生還工程と帰着確認

このシナリオで壊さないもの

  • 夜明けまでという期限
  • 鐘灯の破損と修理工程
  • 落下・炎上・灰霧による本物の死
  • 同行NPCが一人で全部を解決しないこと
  • ゼロット演出であっても、距離・怪我・崩落・時間はごまかさないこと
この一本の主な満足点は「世界を歩いた感じ」と「一緒に帰ってくる感じ」である。 ただし死亡許容度は極なので、吊橋、落下、灰霧、鐘灯暴走、崩落などの危険は本当に死へ届く。 啓示で運用するため、致死の手前には必ず見えている徴候と回避の筋を置く。

2. プレイヤー向け導入

導入文（そのまま読んでよい）

秋の終わり、谷あいのベルグレン村は、本来なら「星灯祭」の最後の夜を迎えているはずだった。 だが村の中央に据えられた星護の鐘灯は、雷を浴びて砕け、灯守の老人はその場で倒れた。 村人たちは口を揃えて言う。あの灯が夜明けまでに山上の空鐘修道院で継がれなければ、 谷の奥で眠る灰眠りの竜は、今年は眠ったままではいてくれない、と。

老灯守の弟子である少女リネットは、震えながらも、壊れた鐘灯の籠を背負って言う。 「……わたし、道と手順は知ってる。だから、あそこまで行ければ、きっと継げる。 でも、ひとりじゃ無理。途中で、あれを守って、運んで、直して、鳴らしてくれる人がいるなら」

かくしてあなたたちは、夜明けまでの数刻で、月鈴峠を越え、空鐘修道院へ向かうことになる。 行く手には崩れた巡礼路、夜の市、灰を吸う獣、そして、まだ鳴っていないはずの鐘の音が待つ。

プレイヤーに最初から見せてよいこと

  • 星護の鐘灯を夜明けまでに継げなければ、村へ大きな災厄が来る。
  • 同行するリネットは手順の継ぎ手であり、単純な護衛対象以上の「必要人物」である。
  • 月鈴峠には複数の道があり、それぞれ魅力と価格が違う。
  • 山道では「鐘」や「名」や「約束」に関わる奇妙な現象が起こる。
  • 最後は灯を継ぐだけで終わらず、帰る・逃げる・誰を確かめるかまで問われる。

3. GM向け要約

シナリオの問い

PCたちは、壊れた星護の鐘灯を夜明けまでに空鐘修道院で継ぎ、リネットを含む誰をどう生きて帰らせるか。

クリア条件

  1. 星護の鐘灯、またはその代替となる灯の核を山上まで持ち込む。
  2. 空鐘修道院で継ぎの儀を完了し、村を守る鐘火を灯す。
  3. 少なくとも一人が鐘火成立を村へ届く形で生還または確認させる。

失敗条件

  1. 夜明けを迎えても鐘火が成立しない。
  2. リネットが死亡し、代替手順も成立しない。
  3. 鐘灯の核が完全破壊され、代替素材もなく、儀式を始められない。

別利益

  • リネットを無傷に近い形で帰す
  • 鐘灯の古い器を壊さずに修復する
  • 途中の共同体や異形たちと敵対せず帰る
  • 星火の欠片や鐘の記憶を持ち帰る

極運用メモ

このシナリオは死亡許容度が極である。したがって、落下、灰霧窒息、崩落、鐘火暴走は本当にPC死亡へ届いてよい。 ただし 適正 + 啓示 なので、致死要因は必ず事前徴候を見せること。吊橋なら軋み・切れかけた綱・古い警告板。 灰霧なら鳥の死骸・咳き込む獣・視界の白濁。鐘火暴走なら白熱亀裂・音の濁り・器の歪み。見えていた / 読めた / 仕込めた のいずれかが無いまま致死へ飛ばしてはいけない。

4. 想定PCとフック

PC3人の役割は固定しないが、次の三つが揃うと扱いやすい。 王道ファンタジーらしく、前で受ける役、道を開く役、祈りや術で支える役 のうち少なくとも二つがあると、 共同作業と旅の味が太くなる。

前衛 / 守り手

  • 壊れた鐘灯やリネットを守る理由がある
  • 落下・足場・時間稼ぎに強い
  • 「帰す」ことに責任を感じやすい

技巧 / 道開き

  • 近道や裏道、罠、修理、搬送に強い
  • ルート差を一番楽しめる役
  • 鐘灯の補修や橋越えで輝きやすい

祈り / 術 / 知識

  • 鐘の伝承、修道院、怪異、祝詞に関われる
  • リネットの手順不足を補う余地がある
  • 最終儀式の成否へ絡みやすい

おすすめの関係フック

  • リネットの父である老灯守に借りがあった
  • 星灯祭で名を上げたい / 役に立ちたい
  • 空鐘修道院の遺物や鐘に私的な興味がある
  • 「誰かを無事に帰す」約束を一度破ったことがある

PCどうしの共通関係の置き方

次のどれかを選ぶと導入が早い。

  • 同じ村から頼まれた冒険者仲間
  • 祭りの警護・搬送・奉仕で集まっていた寄り合い
  • 別件でベルグレン村へ来ていたが、老灯守の死を見て引き受けた一行

6. 舞台と既知情報

舞台の輪郭

ベルグレン村は、月鈴峠の麓にある小さな宿場村である。畑と果樹と灰白の石壁の村で、 年に一度の星灯祭では、山上の空鐘修道院へ灯を継ぎに行くのが古いならわしになっている。 村の人々は、竜や鐘の伝承を全部信じているわけではない。だが「継がない年には良くないことが起きる」 という程度には、誰もがそれを知っている。

PCが知っていてよい常識

  • 空鐘修道院は今は廃されているが、継ぎの夜だけは開く。
  • 月鈴峠には複数の道があり、巡礼街道は安全、玻璃谷は近道、林檎段丘は地元道だ。
  • 鐘や祠の前では、名前と約束を軽んじると良くない、という俗信がある。
  • 灰霧は夜明け前に濃くなりやすく、長く吸うと命に関わる。
  • 鐘灯の器が壊れていても、灯の核が無事なら継ぎの可能性は残る。

通りやすい手段

  • 村からの正式依頼状や祭具を示して正規に通る
  • 地元の作法に従って贈り物や挨拶をする
  • 共同作業として「誰が守り、誰が運ぶか」を先に決める
  • 鐘や名に関する奇妙さを、笑って捨てずに受け止める

起こりやすい代償

  • 負傷、落下、疲労、灰霧による呼吸障害
  • 村や道の共同体への借り
  • 時間の消費
  • 鐘灯の傷み、器具の破損

起こりやすい余波

  • 鐘の記憶に名を覚えられる
  • 地元共同体との関係が変わる
  • 夜市や修道院に再訪したくなる持ち帰りが残る
  • 「誰を先に帰すか」が次の関係線になる

ゼノス向けの見どころ

  • 越境の魅力: 谷の灯、月下に青く鳴る石橋、玻璃谷の結晶崖、林檎段丘の夜市、空鐘修道院の天空回廊。
  • 文化差の価格: 鐘の前では名を名乗る、夜市では先に小さな贈り物を置く、修道院では左手で火を扱わない。
  • 回復点: 青鐘の湧き泉。冷たいが、灰を洗い流し、鈴のような余韻を胸に残す。
  • 畏敬点: 雲見の尾根。谷の灯が星と見分けがつかない高さに見える。
  • 偶発フックの母材: 後夜祭の旅芸人、撤収する夜市、山羊飼い、検問の修道士、風に鳴る名もない小鐘。

7. シナリオ骨格

夜明け時計
このシナリオでは、公開の期限として 夜明けまで4刻 を置く。 大きな遠回り、長い足止め、失着からの立て直しで1刻進めてよい。 4刻に達した時点で鐘火が成立していなければ、基本的に失敗条件へ触れる。
見えている未解決問題 5 共有ボトルネック 3 主要局面 5 帰路も別工程

第1局面　ベルグレン村、消えた祭り灯

難度: 普通 時計: 0→1刻の可能性

局面の問い: 何を持って出発し、どの道を選び、だれと何を一緒に背負うか。

GMが提示する材料

  • 老灯守の死と、夜明けまでに継がねばならないという現状。
  • 壊れた星護の鐘灯。器は割れているが、灯の核は弱く残っている。
  • 三つの道の概要。
  • 村で用意できる支援（依頼状、祝油、麻縄、簡易担架、林檎酒、鐘守の古地図など）を2〜4点。

ここで前に出る未解決問題

  1. どの道で行くか
  2. 誰が鐘灯を運ぶか
  3. リネットをどう守るか
  4. 何を持参し、何を捨てるか
  5. 夜明けまでの時間見積もりをどう取るか

ここで得られるものの例

  • 新しい手札: 村長の依頼状、祝油、補修釘、鐘守の地図
  • 新しい縁: 白鐘修道会への紹介、山羊飼いの目撃情報
  • 新しい余韻: 祭りの途中で止まった舞台、歌いかけのまま終わった子守歌

第2局面　月鈴峠への三つ道

難度: 普通〜困難 ゼノス主担当

局面の問い: どのルートを通り、何を旅の収穫として拾うか。

巡礼街道（安全ルート）

魅力: 石畳と鐘柱、修道士の休所、青鐘の湧き泉。

価格: 検問、奉納、やや遠回り。1刻進む可能性が高い。

通りやすい手段: 村の依頼状、礼儀、祝油の奉納。

得やすい収穫: 正規の鐘律、銀の小鈴、湧き泉での回復。

玻璃谷（近道）

魅力: 月を返す結晶崖。最短距離で修道院前へ近づける。

価格: 足場、落下、灰獣。鐘灯を守りながら進むのが難しい。

通りやすい手段: 麻縄、前衛の支え、軽量化、段取り。

得やすい収穫: 星鉄の欠片、強い景色、鐘灯補修材。

林檎段丘（地元ルート）

魅力: 夜市、地元の水路、祭りの残り火。旅情がいちばん濃い。

価格: 作法違い、贈り物、言葉の綾。地元の礼を外すと面倒が増える。

通りやすい手段: リネット同行、林檎酒や小銭の贈り物、名乗り。

得やすい収穫: 夜市の品、裏道、鐘の噂、あとで効く奇妙な縁。

ルートごとの代表イベント

  • 巡礼街道: 白鐘修道士が「なぜ継ぐのか」を問う。正規の鐘律や護符を得られるが、急ぐと検問が重い。
  • 玻璃谷: 結晶崖の反響で鐘灯の核が暴れかける。補修材を得られるが、落下と破損の危険が高い。
  • 林檎段丘: 後夜祭の夜市で「小さな贈り物」と「名乗り」の礼を試される。奇妙な有利を得やすいが、不作法は後半の価格になる。

ゼノス・モードの返し方

ここでの失敗は、ただ遅れるだけでなく、別の縁や別の持ち帰り へ返す。 たとえば巡礼街道で足止めされたなら、待つ間に古い鐘守の歌を聞ける。 玻璃谷で落石に追われたなら、逃げ込んだ窪地で星鉄を拾える。 林檎段丘で礼を間違えたなら、怒った夜市の主が「借り」を刻むが、あとで裏道を見せてくれる。

第3局面　祈りの弦橋と霜名の足場

難度: 困難 ヨーク主担当 落下は致死へ届く

局面の問い: 誰が何を肩代わりし、どうやって鐘灯と同行者を対岸へ渡すか。

修道院手前の谷には、古い吊橋「祈りの弦橋」が残っている。綱は半ば朽ち、 風に鳴る細い音が、まだ生きている橋なのか、もう死んでいる橋なのか分からない。 月光が差すと、板や綱に過去の巡礼者の名が青く浮かび、踏み出すべき足場を示すことがある。

見えている危険

  • 綱の劣化。重い荷や無理な同時移動で切れる。
  • 灰霧が谷底から巻き上がり、呼吸を奪う。
  • 鐘灯の核が橋の共鳴で暴れ、器がさらに割れる可能性。

ここで効く共同作業

  • 二人以上で運ぶと安くなるもの: 鐘灯 / リネット / 補修材
  • 肩代わりの窓: 灰霧の吸い込み、恐怖、荷重、時間稼ぎ
  • 帰着確認の先取り: 橋を渡り切った後、誰を一番先に数えるかを宣言させる
見えていた / 読めた / 仕込めた のヒント
  • 第1局面で麻縄や担架を用意していれば明確に効く。
  • 第2局面で得た鐘律・星鉄・夜市の品は、橋の共鳴や補修へ返せる。
  • 橋そのものへ礼を尽くす、名を呼ぶ、誰かのために渡ると宣言するのは、人情 + ゼロットに噛み合う。

この局面での失敗返し

  • 通常の失敗: 時間消費、負傷、鐘灯損傷、対岸へ持ち込める物の減少
  • 重い失敗: 落下、灰霧窒息、リネットの離脱、鐘灯核の暴走
  • それでも残すもの: 名の浮いた足場、橋が示した「帰したい相手」の輪郭、犠牲による進路確保

第4局面　空鐘修道院、継ぎの儀

難度: 過酷 最終工程 三役分担

局面の問い: 壊れた鐘灯をどう継ぎ、誰が何を支払い、灰竜の目覚めより先に鐘火を成立させるか。

山上の空鐘修道院は、半ば崩れた回廊と、雲の上へ突き出た鐘楼だけが残っている。 ここで必要なのは単純な点火ではない。器の補修、鐘律の成立、継ぎ手の保持 の三つを同時に守らねばならない。 しかも灰眠竜の影は、鐘が鳴る前から塔の外壁を這い上がり始める。

儀式の三役

  1. 補修役: 器を星鉄・釘・祝油などで持たせる。足りなければ自分の大事なものや装備を代用素材にしてもよい。
  2. 鳴らし役: 正しい順で鐘律を成立させる。誤ると灰竜の影を強く引く。
  3. 保持役: リネット、灯の核、塔の足場、仲間の退路のどれかを守る。

この局面で前に出る別利益

  • リネットの心と体を壊さない
  • 古い器を残して継ぐ
  • 塔を完全崩壊させずに済ませる
  • 村へ持ち帰れる証拠や欠片を残す

ゼロットが通りやすい窓

  • まだ鳴っていない未来の音を、順番のヒントとして扱う
  • 名の浮く霜を、足場や守るべき順番の象徴として使う
  • 「鐘楼が証人になる」「誓いが音として残る」などの詩的な言い換えを、危険無効化ではなく儀式成立の演出として使う
代替成立のルール

補修材や鐘律が足りないとき、GMは 大事なもの / 装備 / 名声 / 関係 のいずれかを 重く代償にすることで、一段だけ成立を引き寄せてよい。 ただし、それで消した危険は必ず別の傷跡として残すこと。 死亡許容度が極なので、「自分が保持役になる」宣言は、そのまま致死の最前線に立つことを意味してよい。

第5局面　灰眠竜の影と帰着確認

難度: 過酷〜絶望 生還工程

局面の問い: 鐘火成立のあと、だれをどの順で帰し、何を村へ届く形で残すか。

鐘火が成立すると、灰眠竜そのものが起きるのではなく、まず「影」が暴れる。 塔の壁を割り、階段を崩し、火の届く外縁まで逃れた命を試す。ここで大事なのは、 ただ逃げることではない。鐘火が本当に届いた証 を持ち帰るか、 塔に誰かが残って最後の一打を続けるか の判断が最後に問われる。

帰着確認

ヨーク・モードでは、ここで各PCに「誰を数えて帰るか」を言わせると強い。 自分だけではなく、誰の無事を確かめたいか を言葉にした時点で、関係線が確定する。 

【帰着確認】
最初に数える相手:
その相手が欠けていたら、何を捨ててでも探しに行くか:
村へ持ち帰る証:
もう持ち帰れないと諦めるもの:

ここで起きうる結末の分岐

  • 全員帰還: もっとも明るい結末。人情寄りの余韻が強い。
  • 鐘火成立 + 一部退場: 村は守られるが、旅は傷を残す。
  • 鐘火成立 + 村への証だけ届く: 誰かが山に残ったままでも、物語としては成立する。
  • 鐘火未成立: 村へ大きな災厄。だが何を持ち帰ったかで、次の物語の火種は残せる。

9. GM運用メモ

最初の投稿で出してよい三案

  1. まず支援を固めて巡礼街道を上る
  2. 補修材を見込んで玻璃谷の近道を急ぐ
  3. リネットを頼りに林檎段丘の地元道を行く

啓示なので、最初の投稿ではこの三案を「出発点」として見せてよい。 ただし最適解とは言わない。各案の価格と魅力を並べ、PCの個性と守りたいものから選ばせる。

余波の向き（人情）

  • 成功しても借りや情けが残る
  • 共同体の目、約束、だれを先に帰すかが前に出る
  • 敵意より「返すべきもの」「見捨てた負い目」で返す

極でも理不尽にしないための見せ方

  • 落下: 軋み、剥がれ、風、古い警告板
  • 灰霧: 白んだ視界、咳き込む山羊、湿った布の必要性

  • 崩落: 小石、きしみ、先に落ちる金具や粉塵

10. エンディング

大局的勝利

  • 鐘火成立
  • リネット生存
  • 村へ鐘火成立が届く
  • 主要PC全員または全員の帰着確認ができる

村は守られ、星灯祭は「今年だけは違う形」で終わる。旅履歴ももっとも豊かに残る。

成功だが痛みあり

  • 鐘火成立
  • 誰かが残る、死ぬ、壊れる、あるいは大切なものを失う

村は守られる。だが帰り道に数える名前が一つ足りない。人情の余韻を太くする結末。

痛み分け

  • 鐘火は不完全、または村を守る範囲が足りない
  • しかし何かしらの証・縁・欠片は持ち帰る

単発で閉じるなら苦い終わり。続けるなら強い火種になる。

失敗

  • 夜明けまでに鐘火が成立しない
  • 継ぎ手も核も失い、村へ大災が向かう

極なので、ここは本当に重くてよい。ただし何が原因で、どこで手遅れになったかは明確に示す。
