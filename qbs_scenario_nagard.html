風哭きの塔と帰り道の鐘

角獣の大移動を逸らすため、辺境の村から古い鐘塔へ向かう旅。安全ルート・近道・地元ルートの読み、異境の作法、肩代わりと共同作業、そして少しだけ詩的で変な理屈が通る森を通って、何を持ち帰り、誰と帰るかを問う一夜のワイルダネスアドベンチャーです。

剣と魔法の辺境ファンタジー JRPG的中世ヨーロッパ風 ゼノス / ヨーク / ゼロット 人情主軸 死亡許容度: 極

1. シナリオ概要

題材パック王道冒険 + 辺境護送 + 森林越境 + 祭礼前夜
舞台フレーム辺境村ルーデン、月迷いの森ルスヴァルト、風哭きの塔ベルグロック
行動フレーム踏破、救出、ルート選択、奇習との交渉、共同作業による鐘塔起動
脅威フレーム時間切れ、山野の事故、魔獣の群れ、土地固有の作法違反、帰路断絶
代償フレーム疲労、負傷、補給不足、借り、約束、共同体への顔向け、帰り道の喪失
終了フレーム塔へ到達して終わりではなく、鐘を鳴らし、角獣渡りを逸らし、帰路の証を持って帰るところまで問う
シナリオの問い次の夜明けまでに風哭きの塔の《帰り鐘》を鳴らし、角獣渡りを村と街道から逸らせるか。
想定プレイ時間感PBMなら3〜5局面、卓上なら4〜6時間前後の濃度を想定

ハイコンセプト

もしも、帰り道を知る者だけが鳴らせる鐘が沈黙し、翌朝には魔獣の大移動が村と街道を呑むとしたら。

PCたちは、未知の森に触れつつ、異境の理屈と現実の危険の両方を抱えて、鐘塔へ向かうことになります。

このままだと失うもの

  • ルーデン村の冬備えと家畜
  • 街道の隊商路と近隣集落の補給線
  • 鐘守とその弟子の生還可能性
  • 村が長年守ってきた「帰れる土地」という感覚
狙いどころ: 一本道の突破劇ではなく、安全ルート / 近道 / 地元ルート の違い、異文化の読み、同行者との共同作業、そして「何を持ち帰る旅にするか」を太くする設計です。

ゼノスの効きどころ

森の作法、月梢の小夜市、帰り鐘の伝承、持ち帰り候補を毎局面で返しやすくしています。失敗は停止より枝分かれへ寄せます。

ヨークの効きどころ

同時に背負うと軽くなる工程、肩代わりの窓口、帰着確認を明示しました。NPCは媒介役であり、PC同士の連帯そのものは命令しません。

ゼロットの効きどころ

詩的・言語遊戯と夢幻・異世界自然化を歓迎枠にします。ただし時間・落下・凍傷・角獣渡り・鐘の起動条件は壊しません。

- 詩的・言語遊戯
- 夢幻・異世界自然化

【残す一貫性】
- 期限は本当に切れる

3. 公開用30秒ピッチ

タイトル: 風哭きの塔と帰り道の鐘

キャッチコピー: 「帰る理由」を知る者だけが、森の向こうの鐘を鳴らせる。

30秒ピッチ: 冬前の辺境村ルーデンでは、翌朝に角獣の大移動が迫っている。群れを逸らす《帰り鐘》を鳴らせるはずの鐘守は森で消息を絶ち、古い塔は沈黙したまま。PCたちは冒険家ナガード・ミニオンとともに、月迷いの森ルスヴァルトを越え、異境の作法と旅の代価を抱えながら、風哭きの塔を目指す。

最初の謎: なぜ鐘守は戻らなかったのか。なぜ《帰り鐘》は「帰る理由」を問うのか。

最初の賭け金: 村と街道の冬越し、失踪者の生死、そしてPCたち自身の帰路。

感情の約束: 景色と奇習に目を奪われつつ、最後には「誰と帰るか」を静かに問う。

4. 入口の6点とPC前提

入口の6点

  1. あなたは誰か: 辺境で依頼を受ける冒険者、護衛、狩人、神官、術師、旅芸人、学徒など。
  2. 今なにが起きているか: 次の夜明けに角獣渡りが始まる。鐘守は行方不明。《帰り鐘》が鳴らなければ村と街道が危ない。
  3. まだ何が分からないか: 鐘守の所在、塔の現状、森の近道がまだ生きているか、鐘を鳴らす正しい手順。
  4. この事件のどこが特別か: ルスヴァルトでは、ただ進むだけではなく「帰る理由」を問われる。
  5. このままだと何を失うか: 村の冬備え、街道の補給、失踪者の命、PCたちの帰り道。
  6. どんな気分の物語か: 旅情と人情のある冒険だが、足を滑らせれば本当に死ぬ。

選びやすいPC立場例

  • 辺境村で顔を売りたい若い剣士 / 傭兵
  • 森や山道に詳しい狩人 / 斥候
  • 風と祈りを扱う神官 / 精霊術師
  • 荷運びと野営管理に慣れた冒険者 / 商隊護衛
  • 古塔や古歌の伝承を追う学徒 / 旅の記録者

今回の野望例: 「ルーデン村を守る」「帰り鐘を鳴らして生還する」「月迷いの森の秘密を持ち帰る」

PC同士の共通関係のおすすめ

  • 同じ依頼を受けた即席パーティ: 一番乗せやすい。ナガードが段取り役になりやすい。
  • 同じ隊商路に借りのある顔見知り: 人情と帰路の重みが強くなる。
  • 村に一度は世話になった旅人たち: 村を見捨てにくく、人情主軸に噛み合う。

5. 同行者NPC ナガード・ミニオン

基本プロフィール

名前: ナガード・ミニオン

立場: 辺境道を渡り歩く中堅冒険家。地図・補給・宿の手配に強い。

第一印象: 荷を数え、天気を見て、空気が荒れそうなら真っ先に場を丸める。強く押すより、皆が引ける落としどころを探す。

性格: まとめ役 / 安定志向 / マッチャー気質 / 小市民的 / 事なかれ主義。

機能

  • 旅の案内: 外縁の道、宿場、天候の読み、補給の感覚。
  • 共同作業の媒介: 「誰が前に出て、誰が後ろを支えると楽か」を言語化してくれる。
  • 余波回収: 村人との説明、宿駅との交渉、荷の整理、負傷者の応急処置。

ナガードがいると安くなること

  • 安全ルートの手配と野営段取り
  • 村人や宿駅との交渉の初動
  • 「いったん落ち着こう」の空気作り

ナガードが抱える弱み

  • 危険が見えると、達成より生還を優先したくなる
  • 被害を最小化しようとして、時に決断が遅れる
  • 「全員が少し損をしてでも丸く収める」方向へ寄りがち
使い方のコツ: ナガードはヨークそのものではなく、PC同士が誰と誰を組ませると動きやすいかを見える化する媒介役です。彼が全部を解くのではなく、PCの連帯線を整えるために置いてください。 
〔関係: ナガードの道中帳｜長期: 宿駅・補給・外縁ルートの交渉で背中を押す / 置き忘れ・水濡れ・焼失で失う〕
〔手順: ナガード流の野営分担｜中期: 休息・見張り・負傷者搬送を一宣言で扱いやすい / 仲違い・混乱・人数欠けで鈍る〕
【感情: ナガードの弱腰への苛立ち｜短期: 急ぐ場面で意思統一が高くつく / 対話・成果・肩代わりで変質】

6. プレイヤー公開情報

PCが最初から知っていてよい常識

  1. ルーデン村は毎年、風哭きの塔の《帰り鐘》で角獣渡りを逸らしてきた。
  2. 月迷いの森ルスヴァルトでは、節目の道標に「自分が何を持って帰るか」を口にすると道に嫌われにくい。
  3. 森には安全ルート・近道・地元ルートがあり、速い道ほど事故が増え、地元ルートほど作法を外すと痛い。
  4. 鐘守エドリクと弟子のミラは昨日の昼に出発し、今も戻っていない。
  5. 月の高い夜、森の奥で小さな市が立つという噂がある。そこで得た物は「帰り道」に縁を持つらしい。

通りやすい手段

  • 地形読み、野営、斥候、魔物避け、荷の軽量化
  • 作法に合わせた声かけ、贈り物、道標への応答
  • 分担による共同作業、肩代わり、負傷者の搬送

難しい / 不可能なこと

  • 森の作法を踏み倒して最短で無傷踏破すること
  • 塔に着いただけで鐘が自動的に鳴ること
  • 極限状態の無理押しをノーコストで帳消しにすること

安全ルート

灰橋川沿いの旧街道

遅いが野営しやすい。橋崩れや荷の搬送が課題になりやすい。村人受けは良い。

近道ルート

牙尾根の獣道

速いが落石・転落・ハーピィ・冷気に弱い。時間を買う代わりに肉体の代償が増えやすい。

地元ルート

木霊の回廊

作法と贈り物が要る。うまく通れば近くて静か。失礼をすれば迷いや誤解の余波が立つ。

7. GM用真相

真相の要点: 《帰り鐘》は「外敵を追い払う鐘」ではなく、この土地へ帰る者たちの気配を濃くして、角獣渡りを谷向こうへ流す鐘です。だから鳴らすには、単なる力よりも「帰る理由」と「帰り道の証」が要る。

鐘守失踪の真相

鐘守エドリクは事故で重傷を負い、塔の手前の岩屋に身を寄せている。弟子ミラは彼を助けようとして月梢の小夜市へ助けを求めに行き、森の作法に足止めされている。

エドリクは鐘の正しい鳴らし方を知るが、一人では帰れない。

鐘の正しい手順

  1. 塔の足元で帰路を示す火を焚く
  2. 塔上で帰りの詞を読む
  3. 綱・留め具・鐘木を同時に扱う

正しい手順を知らなくても強引に鳴らすことはできるが、大切なもの / 手札 / 帰り道の証 の破壊や、誰かの取り残しといった重い代償が必要になる。

このシナリオで重く扱う失敗条件

  • 次の夜明けまでに鐘が鳴らない
  • 帰路が完全に断たれる
  • 塔での無理押しによって全員が行動不能 / 置き去りになる
  • 角獣渡りの進路を村側へ向けたままにしてしまう

作法違い、寄り道、誤解それ自体は即失敗にしません。誤解、借り、別の案内人、余計なひと手に変換してください。

8. 主要局面

SCENE 1

ルーデン村の依頼と出立

局面の問い: 何を持って森へ入り、どのルートで塔を目指すか。

冬前の灰空。村では荷車が壁代わりに並べられ、家畜小屋の前で人々が不安そうに空を見ている。鐘守の家は開け放たれたまま、帰り支度の途中で時間が止まっている。

見せる情報

  • 時間切れは次の夜明け
  • 鐘守と弟子は未帰還
  • ルートは3本。村人は安全ルート推し、ナガードも基本はそこを勧める
  • 森の道標に「持って帰るもの」を告げる習わし

起こりやすい代償

  • 時間を取れば準備は整うが、夜移動が増える
  • 荷を欲張ると道中の事故と疲労が増える
  • 村人への態度で後の援助 / 噂が変わる

ゼノス / ヨーク

  • 出立前に「何を持ち帰りたいか」を一言ずつ置かせると旅履歴が太い
  • 同行メモをここで決めると以後の肩代わりが自然になる

置きやすい返り

  • 〔関係: ルーデン村の貸し〕
  • 〔資産: 角獣避けの火粉〕
  • 〔手順: 出立前の分担表〕
SCENE 2

月迷いの森の分岐

局面の問い: 安全・速度・地元の作法のどれを優先して進むか。

森へ入ると、風向きに逆らって木の葉が道を指し、古い道標には擦れた文字が残っている。言葉をかけると、道標の向きがわずかに変わる。

安全ルート: 灰橋川沿い

橋崩れ、荷の搬送、ぬかるみ。遅いが救助と補給に強い。村人への顔向けが良く、後の援助が残りやすい。

近道ルート: 牙尾根

速い。落石、滑落、冷気、ハーピィ。時間を稼げるが、死亡許容度: 極 を感じやすい。

地元ルート: 木霊の回廊

「持ち帰るもの」を正しく告げ、小さな贈り物を道標に置けば通りやすい。失礼をすると迷いではなく、誤解・借り・別の案内人 が立つ。

ナガードの反応

安全ルートなら手際よく段取りを組む。近道には露骨に顔をしかめる。地元ルートでは「作法だけは雑にやるな」と真顔になる。

ゼロット窓: 道標や森への声かけは、詩的・言語遊戯が最も通しやすい場所。たとえば「月明かりを借りて帰り道の輪郭だけ先に持ち帰る」のような宣言は歓迎できるが、落下や迷子の危険は消さないでください。
SCENE 3

月梢の小夜市 / あるいは無人の野営地

局面の問い: 異境の作法に触れ、鐘塔へ行くための「正しい知識」か「別の足場」を得られるか。

月が高ければ、水面や濡れた石の上にだけ小さな市が立つ。遅れたり作法を外したりした場合は、市ではなく鐘守の古い野営地へ流れ着く。どちらでも、帰り鐘の真実に近づく材料は得られる。

月梢の小夜市で得られるもの

  • 弟子ミラの所在
  • 《帰り鐘》の詞の断片
  • 〔素材: 月梢市の木札〕
  • 回復点としての温かい木の茶

野営地で得られるもの

  • 鐘守の手帳と不完全な手順
  • エドリクの血痕と避難先の推測
  • 〔証左: 鐘守のメモ〕
  • ただし回復は薄く、時間がきつい

人情の返り

  • 市の人々は金より「帰る理由」の話を欲しがる
  • ミラは鐘守より先に村を心配している
  • ナガードはここでようやく本音を少し漏らす

ナガードの小さな告白

「俺は派手な成功談より、ちゃんと宿に戻れた話のほうが好きなんだ」とこぼす。彼が何を“持ち帰りたい人間”かが見える。

SCENE 4

鐘守エドリクの岩屋

局面の問い: エドリクをどう扱い、正しい手順・鐘木・帰路の証をどう整えるか。

塔の直前、風を避ける岩屋に鐘守エドリクが倒れている。足は潰れ、単独では動けない。彼は鐘の正しい鳴らし方を知るが、手早く動かなければ夜明けに間に合わない。

ここで問う選択

  • エドリクを安全圏へ下ろすか
  • エドリクの手順をその場で聞くか
  • 鐘木や留め具を持って塔へ急ぐか

別利益

  • 鐘守の生還
  • 弟子ミラとの再会
  • 正しい手順の完全習得
  • 帰り道の証を壊さずに済ませること

ヨークの主戦場

誰が負傷者を支え、誰が塔へ先行し、誰が帰路の準備を持つか。一緒に背負うと軽くなる をいちばん見せやすい場面です。

置きやすい価格

  • 【疲労】
  • 【足場不安】
  • 〔共有手札: 役割分担の型〕
  • 【借り: 鐘守を見捨てなかった】あるいはその逆
SCENE 5

風哭きの塔と《帰り鐘》

局面の問い: 誰がどこを担い、何を支払い、鐘を鳴らして帰路を残すか。

塔の周囲には、谷を埋める角獣渡りの唸りと、夜明け前の冷気が満ちている。鐘は裂け目だらけで、綱は凍りつき、足場は不安定だ。

必要工程

  1. 塔下で帰路を示す火を維持する
  2. 塔上で帰りの詞を読む / 代用品でこじ開ける
  3. 綱・留め具・鐘木を同時に扱う

正攻法の利点

手順を揃えていれば、大切なものや手札を壊さずに鳴らせる。代わりに時間と分担と静かな集中が要る。

強引な方法

鐘木の代わりに大切なものを砕く、誰かが危険な位置に留まり続ける、火を自分の荷で増やすなどで起動可能。ただしそのぶん、喪失・取り残し・致命傷が見えてくる。

ナガードの役目

火の番、負傷者の支え、綱止め、撤退の合図のいずれかを埋められる。PCが彼を信じて役割を預けたかどうかで、最後の支えが変わる。

死亡許容度: 極 の使いどころ: ここで退路を確保せず、足場や負傷を無視して無理押しした場合は、落下 / 凍傷 / 群れへの巻き込まれが本当に死へ届きます。啓示卓なので、足場の悪さ・風・時間切れ・帰路の細さは先に明示してください。

9. モード反映メモ

ゼノス・モード

  • 毎回の収穫を 新事実 / 新しい縁 / 新しい手札 / 新しい余韻 のどれかで返す
  • 回復点: 月梢の小夜市
  • 畏敬点: 夜明け前の風哭きの塔
  • 偶発フックの母材: 道標への声かけ、遅配、祭礼の残り香、隊商の噂、市の木札

ヨーク・モード

  • Scene 1 で同行メモ、Scene 4〜5 で肩代わりを前景化
  • 共有で守る対象: 村 / 街道 / 鐘守 / 帰路
  • 共有でしか抜けにくい圧: 搬送、火の維持、綱止め、帰着確認
  • 共有返り: 〔共有手札: 役割分担の型〕 〔共有退路〕 〔二人の名で通る信用〕

ゼロット・モード

  • 歓迎する逸脱: 森や道標や鐘へ詩的に話しかける、夢の論理で「帰る理由」を可視化する、言い回しが儀式になる。
  • 歓迎しにくい逸脱: 共有済みの転落・凍傷・時間切れ・群れの危険を、ノーコストのギャグで無効化する。
  • 支払い先: 火種表示、余波タグ、人物履歴、群衆の注目、奇妙な通じ合い。
  • このシナリオで似合う局所発動: Scene 2 の道標、Scene 3 の月梢市、Scene 5 の帰り鐘の詞。
【局所逸脱の例】
逸脱: 「鐘を鳴らす」のではなく、「塔に帰り道を思い出させる」と宣言する
合図: 詩的・言語遊戯
残す一貫性: 綱・火・足場の工程は消さない
狙う報酬: 失われた鐘の詞の断片を一時的に補う
支払い: 〔火種: うまく言いすぎた帰る理由〕 または 【群衆の注目】

10. 結末と残り方

大成功

正しい手順で鐘を鳴らし、角獣渡りを逸らし、鐘守や弟子も含めて多くを連れ帰る。村には感謝と新しい縁が残る。

痛みのある成功

鐘は鳴るが、帰り道の証、大切なもの、誰かの役目、あるいは誰かの生還を犠牲にする。人情の重みが強く残る。

失敗 / 破綻

鐘が鳴らない、鳴っても進路を逸らせない、あるいは帰路が全滅する。村は避難や損耗の長い後始末へ入る。

残り方の例

Quiddities 《帰り鐘》の真実、森の作法、村と街道が一晩守られた事実、あるいは守れなかった現実
Builds 〔月梢市の木札〕〔ナガードの道中帳〕〔共有退路〕〔帰り鐘の詞〕〔ルーデン村の貸し〕
Stories ナガードとの静かな信頼、肩代わりの負い目、帰れた者と帰れなかった者の差、村へ戻る理由の言い換え

11. 旅履歴と次回フック

旅履歴候補

  • 景: 夜明け前、塔の縁で谷を渡る角獣の白い息
  • 人: 祭りの終わりを知る梟面の市人
  • 風習: 道標へ「持って帰るもの」を告げる
  • 持ち帰り: 月梢市の木札 / 鐘の詞 / ナガードの鍵の傷跡

偶発フック候補

  • 贈り物: 月梢市の市人から届く、次の土地への招待札
  • 面倒: 角獣渡りを逸らしたせいで谷向こうの村から苦情が来る
  • 別ルート: ルスヴァルトのさらに深い古道が地図に浮かび上がる
  • 借り: ナガードが「次はもう少し安全な仕事にしようぜ」と言いながら、また同行を申し出る
一言でまとめると: このシナリオは、「鐘を鳴らせるか」だけでなく、帰る理由を言葉にしながら、誰とどんな帰り道を残すか を選ぶ一篇です。JRPGらしい王道の手触りを保ちつつ、QBSのゼノス / ヨーク / ゼロットが素直に立つよう、旅・共同作業・詩的な理屈を一つの森へ収めています。
