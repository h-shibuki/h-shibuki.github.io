怪盗サックと無銘街の夜鐘

怪盗サック・ライルが盗んだのは、金庫の宝石ではない。下町《背煤区》の「名前そのもの」を街につなぎとめる封印印章です。夜明けまでに返さなければ、その区画は少しずつ誰の目にも留まらない《無銘街》へ落ちていく。すでに看板の字は薄れ、道案内は狂い、巡回兵は家々を見落とし始めました。怪盗を連れて夜の白鐘都を走り、追跡・交渉・潜入・儀式をつないでいく、一晩の都市冒険です。

1. どんな冒険？

ざっくり言うと

怪盗が盗んだ「街の名前」を返しに行く、追跡・聞き込み・潜入・儀式・逃走の一晩です。

今夜のミッション

《無銘印》を取り戻し、法・信・街の三承認を揃えて返印式を通し、背煤区と自分たちの明日を消さないこと。

CRPGでいうと

夜の城壁都市を駆け回る時限メインクエスト。必要な条件を集めながら、追手と時計の両方に追われるタイプです。

この話の気持ちよさ

都市の仕組みを読む手応え、夜だけ開く裏路地の異境感、そして善人ではない怪盗とどう付き合うかの危うさが魅力です。

想定人数3PC想定
舞台白鐘都ベルンハイム、背煤区、名簿塔
同行者サック・ライル（怪盗）
遊び口シティアドベンチャー / 追跡 / 交渉 / 返印儀式

派手な戦闘より、どう通すかが楽しい話です。ただし時計は止まりません。印章の確保、返印式の成立、名簿塔からの離脱はそれぞれ別の山場です。

2. 参加前に知っておきたいこと

  • ベルンハイムでは、区画の名前が正しく結ばれているからこそ、税も祈りも警邏も道案内も機能します。
  • 願訴夜に封印印章を抜くのは最悪に近い禁忌で、区画は「見落とされやすい」状態へ向かいます。
  • 《無銘街》とは、完全に消えるのではなく、「そこにいても気づかれにくい」「守る制度が届きにくい」状態のことです。
  • 返印式には法・信・街の三承認が必要で、どれか一つだけでは安定しません。
  • サック・ライルは大怪盗ではありませんが、裏路地と札と抜け道に妙に詳しい、危ない筋の魔法盗賊として知られています。
  • すでに背煤区では、看板の字が薄れ、道案内が効かず、巡回兵が家々を見落とし始めています。

この話は、ただ追手から逃げるだけの夜ではありません。誰に頼るか、誰を説得するか、何を街に返すかでルートが変わる、都市型のクエストです。

3. どんな主人公がハマる？

映えやすい役割

都市の仕組みを読む人、交渉や手続きを動かす人、追跡・潜入・逃走を担う人のどれかがあると強くハマります。

参加理由の例

背煤区に家族や店がある、市兵や書記局や教会に知人がいる、願訴夜の護衛だった、サックに盗まれた / 助けられた / 借りがある、などが入りやすいです。

4. 同行NPC

サックは有能ですが、善人ではありません。口が達者で、危なくなると平気で人の善意をあてにし、自分が被害者みたいな顔もします。

それでも、裏路地のガイドであり、札剥がしや抜け道や夜の相場勘を持つ、今夜いちばん便利な問題人物でもあります。見張る・使う・背負う・逃がさないのどれを選ぶかで、物語の味が変わります。

5. こんな人におすすめ

夜の都市を走り回るクエスト、盗賊NPCの管理、制度や礼法を使った突破が好きな人におすすめです。派手に殴るより、条件を揃えて、道を開いて、ギリギリで通すのが気持ちいいタイプです。

TRPGが初めてでも、「次に誰へ会いに行くか」「どの手段で通すか」を決めるところから入りやすいシナリオです。
