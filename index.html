Quick Build Saga

PBM形式のやり取りを前提に、プレイヤー向けルール・GM向け資料・世界観ごとの設定集・補助ツールへの入口を整理したトップページです。プレイヤーは「プレイヤーはここから」へ、GMは「GM向け資料」へ進むと辿りやすくなります。

プレイヤーはここからGM向け資料世界観ごとのセクションその他

プレイヤーは クイックスタート → プレイヤー基本ルール → プレイヤー実戦ガイド の順で読むと、最初の参加から実戦的な宣言づくりまで追いやすくなります。PCテンプレート編集ツールと宣言テンプレート編集ツールは、自然言語で考えた案を自分で正式テンプレートへ落としたいとき の厳密入力用です。同じブラウザで使うと保存内容を相互参照できます。

Getting Started

プレイヤーは上から順に見ていけば参加前の準備まで辿れます。GMは 4 から先へ進んでください。

  1. 1クイックスタート / プレイヤー基本ルール遊び方の流れ、宣言、タグ、命運点、成長点と Q/B/S の入口を確認。
  2. 2プレイヤー実戦ガイド / PCテンプレート編集ツール宣言の組み立てを読み、PC情報を整えて下書きを用意。
  3. 3宣言テンプレート編集ツール局面や PC 情報を参照しながら、実際の宣言文を下書き。
  4. 4GM担当なら GMガイド 以降へ運用・設計・キャンペーンの考え方や実例を確認。

FAQQBS

Q/B/Sって何？

QBS では、いま言語上すでに成立しているものを Quiddities、将来に備えて焦点化・保持したものを Builds、それらが他者との関係の中で変質し続ける連なりを Stories と読みます。資産、関係、感情、火種、人物像の違いは、本質の違いではなく、読み味と記録のための表示差です。

FAQ高難度

選択肢例はそのまま選んでいい？

初級〜中級では出発点として使いやすいですが、峻烈 / 悪辣ではそのままなぞるだけだと重い代償や罠になることがあります。退路や守りたいものを一言足す目安を確認できます。
