公開のポイント

このサイトは画像や外部ライブラリに依存せず、HTMLだけで完結しています。 そのため、GitHub Pages にアップロードするとそのまま公開しやすい構成です。

公開後の URL は、ユーザーサイトなら https://あなたのGitHubユーザー名.github.io/ になります。

最初にやること

1

GitHub に ユーザー名.github.io という名前のリポジトリを作る

例: ユーザー名が example なら、リポジトリ名は example.github.io です。

2

このページ一式をアップロードする

index.html、player-guide.html、gm-guide.html と、公開したい追加の .html を同じ場所に置いてください。

3

GitHub Pages を有効化する

リポジトリの Settings → Pages で、公開元を main ブランチの /(root) に設定します。

4

公開URLを開く

数分後に、https://あなたのGitHubユーザー名.github.io/ で表示されます。追加した HTML は自動で一覧に現れます。

ヒント: 今後追加するページは、<title> を入れておくと一覧の表示名がきれいになります。 <meta name="description" content="..."> を入れておくと説明文も拾えます。
